Application Console

Follow the workflow from data import to map review, data quality, launch RF analysis, JSP 101 reporting, and final briefing output.

Primary workflow

Upload + Map Dashboard Upload MOTH/LAMP CSV data, view where collection data was gathered, inspect suitability hexagons, and manage candidate sites. Data Quality Check stale GPS, duplicate files, missing values, zero/zero rows, filter scope, and confidence issues before reporting. Launch RF Analysis Review L1/L2/L5 timelines, spectrum/spikes, pattern-of-life, and recommended or best viable RF timing windows. JSP 101 Report Build the formal report from map findings, scan scope, RF timing, GPS serviceability evidence, caveats, and decision state. Mission Brief Prepare the concise operational briefing after the evidence and report sections have been reviewed.
